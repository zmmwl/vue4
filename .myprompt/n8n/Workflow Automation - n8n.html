AI-Powered Real Estate Lead Responder & Qualifier (n8n Workflow)

 
** Feel free to contact me if you need help implementing (rbreen@ynteractive.com) **

How to Implement

Follow these steps to get your workflow fully running:
 

  1. Sign Up for Twilio
    👉 https://login.twilio.com/
     
    Purchase a phone number with SMS support.
     
    In n8n, set up Twilio credentials and connect them to the Twilio nodes in the workflow.
     
  2. Add Your OpenAI API Key
    👉 https://platform.openai.com/
     
    Get your secret key.
     
    Add your key to the AI Agent and Chat Model nodes using OpenAI credentials.
     
  3. Set Up Google Sheets
    👉 Copy this sheet - https://docs.google.com/spreadsheets/d/1jSlYXSBA9BJvSuMiqFUJo3omztNJtbejNYEopfbGWN8/edit?usp=sharing
     
    In n8n, connect your Google Sheets OAuth2 credentials and update the sheet ID in the Google Sheets nodes.
     
  4. Set Up Supabase for Chat Memory
    👉 https://supabase.com/
     
    Create a free Supabase project.
     
    Go to Database > Connection Info, and copy your PostgreSQL credentials.
     
    In n8n, go to Credentials > PostgreSQL and create a new connection using your Supabase info.
     
    Link this credential in the PostgreSQL Chat Memory field of your AI Agent node to persist and recall chat history per phone number.
     
  5. Customize Buyer Questions (Optional)
    Edit the AI Agent node that asks pre-qualifying questions.
     
    You can change the prompt to ask any questions relevant to your lead funnel (e.g., loan preapproval, desired move-in date, etc.).
     
    ✅ Ready to Launch!
    Once configured:
     
    The workflow listens for a form submission.
     
    Sends an SMS to the buyer to start a chat.
     
    Uses AI to ask key buyer questions.
     
    Stores chat memory in Supabase (so follow-ups stay contextual).
     
    Summarizes and logs the lead to Google Sheets for your agent.
